Logos + your 10 fonts

✓ ALL 10 FONTS FREE · SIL OFL 1.1
Variable fonts? Only Lilex is a true variable font (weight axis). Geist Pixel ships 5 static shape-variants (Square/Grid/Circle/Triangle/Line) — but its siblings Geist Sans & Geist Mono are variable if you want them. The Quarantine set also includes a variable face (Spitz), not shown here. The rest — Aldone, Bmono, Hikaru, Freak Grotesk, Terminal Grotesque, Conta, Grotesk812, Jones, Lab Mono — are static weights.

⚠ Hikaru Mono is a display face — it falls apart at small sizes, so I've kept it large only and used Lilex for the small mono lines.
